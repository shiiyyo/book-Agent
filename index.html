打开创作控制台

请先在项目目录下启动 API 服务，再点击下方按钮。

打开创作控制台

前端地址：http://127.0.0.1:5000（运行 py api_server.py 后生效）

使用步骤

1. 安装依赖 py -m pip install -r requirements.txt

在项目根目录下执行。

2. 配置 .env

在项目根目录创建 .env，填入 DEEPSEEK_API_KEY 或 MOONSHOT_API_KEY。

3. 启动前端服务 py api_server.py

启动后浏览器打开 http://127.0.0.1:5000，在页面内：选择/新建书籍 → 选模型 → 生成全书大纲 → 从大纲创建章节 → 对每章点击「扩写本章」。
